# 알루미늄 (Aluminum) 파이프 TIG 용접 — 챗봇 응답용 지식 문서

## 1. 재질 개요

- 대표 재질: 5052, 5083, 6061, 6063 등
- 열전도율: 강철의 약 5배 — 열을 즉시 빨아간다
- 산화막: 모재 녹는점(660°C)보다 3배 이상 높은 융점(2,050°C) → 먼저 깨야 모재가 녹는다
- **AC 사용 필수** (EP 사이클로 산화막 청정작용)

## 2. 표준 설정값

| 항목 | 권장값 |
|---|---|
| 극성 | AC (교류) |
| AC 발란스 | EN 65~70% / EP 30~35% (청정작용) |
| AC 주파수 | 80~120Hz (좁고 집중된 아크) |
| 텅스텐 | 2% Lanthanated 또는 Zirconiated, Ø3.2mm |
| 텅스텐 끝 | 뾰족 + AC로 자동 형성되는 공 모양 |
| 와이어 | ER4043 (일반) / ER5356 (강도/내식), Ø2.4~3.2mm |
| 차폐 가스 | 100% Argon, 15~18 LPM (두꺼우면 He 25~50%) |
| 백퍼지 | 보통 불필요 |
| 예열 | ≥6mm 두께 시 100~150°C |
| 루트 갭 | 1.6~2.4mm |
| 루트 전류 | 110~160A |
| 필/캡 전류 | 130~180A |

## 3. 산화막 제거 — 가장 중요한 사전 작업

1. **스테인리스 와이어 브러시 (알루미늄 전용)**로 표면을 닦는다
2. 다른 금속에 쓴 적 없는 새 브러시 사용
3. 닦은 뒤 **30분 이내**에 용접 시작
4. 기름은 아세톤으로 먼저 제거 후 브러시
5. 좁은 갭이면 갭 내부까지 브러시가 들어가야 함

## 4. AC 발란스/주파수 이해

- **EP(전극 양극) 사이클**: 산화막 청정작용 / 텅스텐 발열
- **EN(전극 음극) 사이클**: 모재 발열 / 용입
- EP 비율 ↑ → 청정 강함, 텅스텐 빨리 닳음
- EN 비율 ↑ → 용입 깊음, 청정 약함
- 주파수 ↑ → 아크 좁아짐 (좁은 갭/박판에 유리)
- 주파수 ↓ → 아크 넓어짐 (캡 패스 비드 폭 확보)

## 5. 자세별 핵심 포인트

### 2G
- 진행 속도 강철보다 1.5~2배 빠르게
- 토치 상향 5~10°
- 풀이 반짝이며 흔들리기 시작하면 즉시 와이어

### 5G (좁은 갭)
- Lay wire 기법이 표준
- AC 주파수 100~150Hz로 올려 아크 집중
- 12시 구간은 풀이 가장 빨리 자람 → 페달 빠르게 떼기
- 인터패스 65°C 이하 (특히 5xxx 계열)

## 6. 자주 발생하는 불량

| 불량 | 원인 | 즉시 조치 |
|---|---|---|
| Porosity (기공) | 산화막/기름/수분/가스 부족 | 직전 청소, 가스 15 LPM↑, 텅스텐 청결 |
| Tungsten 검게 변색 | EN 부족 / 전류 과다 | EN 65~70%, 전류 ↓, 텅스텐 한 단계 굵게 |
| Lack of fusion (시작점) | 시작 열 부족 / 산화막 잔존 | 예열, 시작 전 1초 아크 켜기 |
| Sagging (처짐) | 진행 정지 / 입열 누적 | 진행 1.5배 ↑, 전류 5~10A↓ |
| Cracking | 5xxx에 ER4043 사용 등 와이어 매칭 오류 | 5xxx 모재에는 ER5356 |

## 7. 신입이 자주 묻는 질문

**Q. 알루미늄은 왜 AC를 써요?**
A. 산화막을 깨려면 EP 사이클이 필요하고, 모재를 녹이려면 EN 사이클이 필요합니다. AC는 둘을 교대로 줘서 청정 + 용입을 동시에 합니다.

**Q. 텅스텐 끝을 뾰족하게 갈았는데 둥글어졌어요. 잘못된 건가요?**
A. 정상입니다. AC를 켜면 끝이 자동으로 작은 공 모양이 됩니다. 그 공이 텅스텐 굵기보다 너무 크면 전류가 과다하다는 신호이니 줄이세요.

**Q. 시작점이 자꾸 안 녹아요.**
A. 알루미늄은 열을 빠르게 빼앗아갑니다. 두꺼운 파이프는 100~150°C 예열, 또는 시작점에서 페달을 1초 더 밟아 풀을 만든 뒤 진행하세요.

**Q. ER4043과 ER5356 차이는?**
A. 4043은 균열 저항 좋고 색이 어둡습니다(일반용). 5356은 강도/내식성 좋고 색이 밝습니다(5xxx 모재 매칭). 5xxx 모재에 4043 쓰면 균열 가능성 있습니다.

**Q. 알루미늄에 백퍼지가 필요한가요?**
A. 일반적으로 불필요합니다. 단, 위생/식품/약품 배관 등 내부 외관이 중요할 때는 아르곤 백퍼지를 합니다.
