# 탄소강 (Carbon Steel) 파이프 TIG 용접 — 챗봇 응답용 지식 문서

## 1. 재질 개요

- 대표 재질: A53, A106, A234 등 일반 구조용/배관용 탄소강
- 열전도율: 중간 (스테인리스보다 높음, 알루미늄보다 훨씬 낮음)
- 산화 민감도: 낮음 — 백퍼지 없이도 작업 가능
- TIG 적용 부위: 주로 루트 패스. 필/캡은 SMAW/FCAW로 마무리하는 경우도 많음

## 2. 표준 설정값

| 항목 | 권장값 |
|---|---|
| 극성 | DCEN (직류 정극성) |
| 텅스텐 | 2% Ceriated / 2% Thoriated, Ø2.4mm |
| 텅스텐 그라인딩 각도 | 60~75° |
| 와이어 | ER70S-2 / ER70S-6, Ø2.4mm |
| 차폐 가스 | 100% Argon, 12~15 LPM |
| 백퍼지 | 일반적으로 불필요 |
| 루트 갭 | 2.4mm (와이어 굵기) |
| 랜드(land) | 1.5~2.0mm |
| 루트 전류 | 85~110A (4" SCH40 기준) |
| 필/캡 전류 | 100~130A |

## 3. 자세별 핵심 포인트

### 1G — 파이프 회전, 토치 고정
- 회전 속도를 일정하게. 분필로 시계 4등분 표시 → 같은 시간에 통과 연습.
- 고정 위치는 12시~1시 사이.
- 키홀이 동전 모양이면 정상.

### 2G — 파이프 수직, 용접선 수평
- 1G보다 전류 5~10A 하향(중력으로 풀 처짐).
- 토치 상향 5~10°.
- 캡은 멀티 스트링거 2~3줄 권장.

### 5G — 수평 고정, 토치 360° 이동
- 상향(uphill, 6시→12시) 진행이 표준.
- 구간별 페달 조절: 6시 낮게, 3시·9시 중간, 12시 약간 높게.
- 기법: Lay wire (초보) → Walking the cup (숙련).

### 6G — 45° 경사
- 자세가 결과의 90%.
- 파이프 12시 위치를 명치~가슴 높이로 세팅.
- Walking the cup 권장.
- 한 바퀴를 한 번에 가지 말고 구간별로 끊어가도 된다.

## 4. 자주 발생하는 불량

| 불량 | 원인 | 즉시 조치 |
|---|---|---|
| Burn through | 전류 과다 / 진행 정지 | 전류 5~10A↓, 진행 속도 ↑ |
| Lack of penetration | 갭 부족 / 전류 부족 / 후진각 과다 | 갭 확보, 전류 ↑, 토치 5~10° 후진각 |
| Tungsten contamination | 텅스텐 접촉 | 즉시 정지 → 재연마 → 오염부 그라인딩 |
| Undercut | 토에 머무는 시간 부족 | 양쪽 토 0.5초 머물기 |
| Crater crack | 끝맺음 부실 | 페달 서서히 내리며 와이어로 채움 |

## 5. 신입이 자주 묻는 질문

**Q. 텅스텐이 모재에 닿았어요. 어떻게 해요?**
A. 즉시 정지. 텅스텐 끝을 그라인더로 재연마, 오염된 비드 부분은 그라인더로 따고 재시작하세요.

**Q. 풀이 검게 그을려요.**
A. 가스 유량 부족이거나 컵에서 모재까지 거리(스틱아웃)가 너무 깁니다. 가스 12 LPM 이상, 스틱아웃 6~8mm로.

**Q. 후진각/전진각 어떻게 잡아요?**
A. 탄소강은 후진각 5~10°가 표준. 너무 누이면 키홀이 안 보입니다.
