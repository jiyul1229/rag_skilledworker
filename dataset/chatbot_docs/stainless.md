# 스테인리스 (Stainless Steel) 파이프 TIG 용접 — 챗봇 응답용 지식 문서

## 1. 재질 개요

- 대표 재질: 304/304L, 316/316L, 듀플렉스(2205) 등
- 열전도율: 탄소강의 약 1/3 — 같은 전류에서 풀이 더 크게 자란다
- 산화 민감도: 높음 — **백퍼지(backing gas) 필수**
- 내식성 유지: 인터패스 온도 175°C 이하, L 등급 와이어 권장

## 2. 표준 설정값

| 항목 | 권장값 |
|---|---|
| 극성 | DCEN |
| 텅스텐 | 2% Lanthanated(파랑) / 2% Ceriated, Ø2.4mm |
| 텅스텐 그라인딩 각도 | 30~45° (뾰족하게, 끝 평탄 0.3mm) |
| 와이어 | ER308L (304용) / ER316L (316용), Ø2.4mm |
| 차폐 가스 | 100% Argon, 10~12 LPM |
| 백퍼지 가스 | 100% Argon, 5~8 LPM **필수** |
| 루트 갭 | 2.4mm |
| 랜드 | 1.5mm |
| 루트 전류 | 70~95A (탄소강보다 10~15A↓) |
| 필/캡 전류 | 85~110A |
| 인터패스 온도 | ≤175°C |

## 3. 백퍼지 작업 절차

1. 파이프 양 끝을 종이 테이프 + 알루미늄 호일로 이중 마스킹
2. 한쪽 끝에 가스 공급 호스, 다른 쪽에 0.5mm 정도 미세 배기구
3. 파이프 부피의 5~7배 가스가 들어가도록 퍼지 (4" SCH40 1m 기준 약 3~5분)
4. 산소계로 50ppm 이하 확인 (가능한 경우)
5. 용접 중에도 5~8 LPM 유지

## 4. 백비드 색상 판정

| 색 | 판정 | 의미 |
|---|---|---|
| 짚색(Straw) / 은색 | ✅ 합격 | 산소 농도 50ppm 이하 |
| 옅은 청색 | ⚠️ 경계 | 산소 일부 침입 |
| 청색 / 회색 | ❌ 불합격 | 산화 진행 |
| 검정 / 녹색 | ❌ 불합격 | 심한 산화, 부식 취약 |

## 5. 자세별 핵심 포인트

### 1G
- 회전 일정, 페달로 미세 조정.
- 백퍼지 누설 점검 우선.

### 2G
- 키홀 기법 표준.
- SCH10 같은 박판은 페달 70% 이하.
- 위쪽 토에 미용착 잘 생김 → 위빙 시 위쪽 더 머무르기.

### 6G (최고 난이도)
- 자세 + 백퍼지 + 입열 관리 종합.
- 6시 구간이 가장 어려움 (풀 처짐 + 백퍼지 빠짐).
- 한 호흡 = 한 와이어 찍기 리듬.

## 6. 자주 발생하는 불량

| 불량 | 원인 | 즉시 조치 |
|---|---|---|
| Black/blue back bead | 백퍼지 부족 / 누설 | 가스 유량 ↑, 마스킹 재시공, 퍼지 시간 ↑ |
| Carbide precipitation | 인터패스 온도 과다 | 패스 사이 식히기, L 등급 와이어 |
| Sagging root (박판) | 입열 과다 | 전류 ↓, 진행 속도 ↑, 와이어 더 자주 |
| Crater crack | 끝맺음 부실 | 페달 서서히 내리며 와이어 채움 |
| Top toe undercut (2G) | 위쪽 토 머무름 부족 | 위쪽 0.5초 추가, 위빙 폭 ↓ |

## 7. 신입이 자주 묻는 질문

**Q. 백퍼지를 왜 꼭 해야 해요?**
A. 스테인리스 뒷면이 공기에 닿으면 크롬이 산화돼 검은 비드가 되고 그 부분은 내식성을 잃습니다. 아르곤으로 산소를 밀어내야 짚색 백비드가 나옵니다.

**Q. 인터패스 온도는 어떻게 재요?**
A. 175°C 온도 크레용 또는 적외선 온도계. 손등을 비드 옆 10mm에 댔을 때 2초 안에 떼야 할 정도면 다음 패스 안 됩니다.

**Q. 304L의 L이 뭐예요?**
A. Low carbon. 탄소 0.03% 이하. 카바이드 석출이 적어 내식성이 잘 유지됩니다. 용접부에는 L 등급을 쓰는 게 안전합니다.

**Q. 텅스텐을 왜 뾰족하게 갈아요?**
A. 스테인리스는 키홀을 정확히 보고 따라가야 합니다. 끝이 뭉뚝하면 아크가 퍼져 키홀이 흐려집니다.
